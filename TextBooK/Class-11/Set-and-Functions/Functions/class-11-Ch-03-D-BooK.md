# GRAPHS OF SOME STANDARD REAL FUNCTIONS

## 1. Identity Function

The function $f: R \rightarrow R: f(x)=x$ for all $x \in R$ is called an **identity function** on $R$.

**Domain**: $\operatorname{Dom}(f)=R$
**Range**: $\operatorname{range}(f)=R$

---

### Example 1:

Draw the graph of the identity function $f: R \rightarrow R: f(x)=x$ for all $x \in R$.

#### Solution:

Here, $f(x)=x$ for all $x \in R$. We can prepare its table as under.

| $x$ | -2 | -1 | 0 | 1 | 2 | 3 |
| :---: | :---: | :---: | :---: | :---: | :---: | :---: |
| $f(x)=x$ | -2 | -1 | 0 | 1 | 2 | 3 |

On a graph paper, we draw $X'OX$ and $YOY'$ as the x-axis and the y-axis respectively. We take 10 small divisions $=1$ unit.

Now, on this graph paper, we plot the points:
$A(-2,-2)$, $B(-1,-1)$, $O(0,0)$, $C(1,1)$, $D(2,2)$, and $E(3,3)$.

Join these points successively to obtain the straight line $ABOCDE$ as the required graph line, given below.

## 2. Constant Function

Let $c$ be a fixed real number. The function $f: R \rightarrow R: f(x)=c$ for all $x \in R$ is called the **constant function**.

For the constant function $f(x)=c$ for all $x \in R$, we have:
**Domain**: $\operatorname{dom}(f)=R$
**Range**: $\operatorname{range}(f)=\{c\}$

---

(i) $f(x)=2$ for all $x \in R$

## 3. The Modulus Function

The function $f$, defined by $f(x)=|x|$, is called the **modulus function**.
$$
f(x) = |x| = \begin{cases} x, & \text{when } x \geq 0 \\ -x, & \text{when } x < 0 \end{cases}
$$
We know that $|x| \geq 0$ for all $x \in R$.

**Domain**: $\operatorname{dom}(f)=R$
**Range**: $\operatorname{range}(f)=[0, \infty)$

---

### Example 3:

Draw the graph of the modulus function, defined by $f: R \rightarrow R: f(x)=|x|$.

#### Solution:

We can create a table of values for $f(x)=|x|$.

| $f(x)=|x|$ | 3 | 2 | 1 | 0 | 1 | 2 | 3 | 4 |

On a graph paper, plot the points $A(-3,3)$, $B(-2,2)$, $C(-1,1)$, $O(0,0)$, $D(1,1)$, $E(2,2)$, $F(3,3)$ and $G(4,4)$. Join these points successively to obtain the V-shaped graph.

## 4. The Greatest Integer Function (Step Function)

For any real number $x$, the symbol $[x]$ denotes the **greatest integer less than or equal to $x$**.

**Examples**:
- $[6.85]=6$
- $[8]=8$
- $[0.536]=0$
- $[-5.8]=-6$
- $[-5]=-5$

The function $f: R \rightarrow R: f(x)=[x]$ for all $x \in R$ is called the **greatest integer function**, or **step function**.

**Domain**: $\operatorname{dom}(f)=R$
**Range**: $\operatorname{range}(f)=Z$ (the set of all integers)

---

### Example 4:

Draw the graph of the greatest integer function: $f: R \rightarrow R: f(x)=[x]$ for all $x \in R$.

#### Solution:

Using the definition of $[x]$, we have:
- $[x]=-2$ for $-2 \leq x < -1$
- $[x]=-1$ for $-1 \leq x < 0$
- $[x]=0$ for $0 \leq x < 1$
- $[x]=1$ for $1 \leq x < 2$
- $[x]=2$ for $2 \leq x < 3$
- $[x]=3$ for $3 \leq x < 4$

The graph consists of a series of horizontal line segments (steps).

![Graph of greatest integer function [x]](https://cdn.mathpix.com/cropped/2025_09_25_d151f9018eef695a9ce3g-32.jpg?height=746&width=794&top_left_y=171&top_left_x=340)

## 5. The Smallest Integer Function

For any real number $x$, the **smallest integer greater than or equal to $x$** is denoted by $\lceil x \rceil$.

**Examples**: $\lceil 3.8 \rceil=4$, $\lceil 6 \rceil=6$, $\lceil 0.76 \rceil=1$, $\lceil -6.2 \rceil=-6$.

The function $f: R \rightarrow R: f(x)=\lceil x \rceil$ for all $x \in R$ is called the **smallest integer function**.

**Domain**: $\operatorname{dom}(f)=R$
**Range**: $\operatorname{range}(f)=Z$

---

### Example 5:

Draw the graph of the smallest integer function $f: R \rightarrow R: f(x)=\lceil x \rceil$ for all $x \in R$.

#### Solution:

Using the definition of $\lceil x \rceil$, we have:

| Interval for $x$ | Value of $f(x)=\lceil x \rceil$ |
| :---: | :---: |
| $-3 < x \leq -2$ | -2 |
| $-2 < x \leq -1$ | -1 |
| $-1 < x \leq 0$ | 0 |
| $0 < x \leq 1$ | 1 |
| $1 < x \leq 2$ | 2 |
| $2 < x \leq 3$ | 3 |

The graph may be drawn as shown below.

![Graph of smallest integer function](https://cdn.mathpix.com/cropped/2025_09_25_d151f9018eef695a9ce3g-33.jpg?height=702&width=901&top_left_y=221&top_left_x=269)

## 6. The Fractional Part Function

Let $f: R \rightarrow R: f(x)=x-[x]=\{x\}$. Here $f(x)=\{x\}$ denotes the **fractional part of $x$**.

**Examples**: $\{2.35\}=0.35$, $\{6\}=0$, $\{-7\}=0$, $\{-0.65\}=0.35$, $\{-3.75\}=0.25$.

**Domain**: $\operatorname{dom}(f)=R$
**Range**: $\operatorname{range}(f)=[0, 1)$

---

### Example 6:

Draw the graph of the fractional part function: $f: R \rightarrow R: f(x)=x-[x]=\{x\}$.

#### Solution:

We have $f(x)=x-[x]=\{x\}$.
- For $0 \leq x < 1$, $f(x)=x-0=x$
- For $1 \leq x < 2$, $f(x)=x-1$
- For $2 \leq x < 3$, $f(x)=x-2$
- For $-1 \leq x < 0$, $f(x)=x-(-1)=x+1$

Plotting these segments on a graph paper, we get the graph as shown below.

![Graph of fractional part function, f(x)={x}](https://cdn.mathpix.com/cropped/2025_09_25_d151f9018eef695a9ce3g-34.jpg?height=553&width=998&top_left_y=259&top_left_x=197)

## 7. The Signum Function

The function $f: R \rightarrow R$ defined by is called the **signum function**.
$$
f(x) = \begin{cases} \frac{x}{|x|}, & \text{when } x \neq 0 \\ 0, & \text{when } x=0 \end{cases}
$$
The function may be simplified as:
$$
f(x) = \begin{cases} 1, & \text{if } x>0 \\ 0, & \text{if } x=0 \\ -1, & \text{if } x<0 \end{cases}
$$
**Domain**: $\operatorname{dom}(f)=R$
**Range**: $\operatorname{range}(f)=\{-1, 0, 1\}$

---

### Example 7:

Draw the graph of the signum function.

#### Solution:

Clearly, we have:
- $x < 0 \Rightarrow f(x)=-1$
- $x = 0 \Rightarrow f(x)=0$
- $x > 0 \Rightarrow f(x)=1$

We may now draw the graph as shown below.

![Graph of signum function](https://cdn.mathpix.com/cropped/2025_09_25_d151f9018eef695a9ce3g-35.jpg?height=374&width=1089&top_left_y=217&top_left_x=102)

## 8. Polynomial Function

A function of the form $f(x)=a_{0}+a_{1} x+a_{2} x^{2}+\dots+a_{n} x^{n}$, where $n$ is a non-negative integer and $a_{0}, a_{1}, a_{2}, \dots, a_{n} \in R$, is called a **polynomial function**.

**Examples**:
- $f(x)=x^{3}-2 x^{2}+5$ is a polynomial function.
- $g(x)=4 x^{2}-x+2$ is a polynomial function.
- $h(x)=2 x-3 \sqrt{x}+5$ is not a polynomial function.

---

### Example 8:

Let $f: R \rightarrow R: f(x)=x^{2}$ for all $x \in R$. Find its domain and range. Also, draw its graph.

#### Solution:

Here, $f: R \rightarrow R: f(x)=x^{2}$ for all $x \in R$.
**Domain**: $\operatorname{Dom}(f)=R$
**Range**: $\operatorname{range}(f)=\{x \in R: x \geq 0\}=[0, \infty)$

Now, we prepare a table of values:

| $x$ | -2 | -1.5 | -1 | 0 | 1 | 1.5 | 2 |
| :---: | :---: | :---: | :---: | :---: | :---: | :---: | :---: |
| $f(x)=x^{2}$ | 4 | 2.25 | 1 | 0 | 1 | 2.25 | 4 |

Now, we plot the points $A(-2,4)$, $B(-1.5,2.25)$, $C(-1,1)$, $O(0,0)$, $D(1,1)$, $E(1.5,2.25)$ and $F(2,4)$. Join these points freehand successively to obtain the required curve (a parabola).

## 9. Rational Function

The functions of the type $\frac{f(x)}{g(x)}$, where $f(x)$ and $g(x)$ are polynomial functions of $x$, defined in a domain where $g(x) \neq 0$, are called **rational functions**.

---

### Example 10:

Let $f:(R-\{0\}) \rightarrow R: f(x)=\frac{1}{x}$ for all values of $x \in R-\{0\}$. Find its domain and range. Also, draw its graph.

#### Solution:

Let $f:(R-\{0\}) \rightarrow R: f(x)=\frac{1}{x}$.
**Domain**: $\operatorname{dom}(f)=R-\{0\}$
**Range**: $\operatorname{range}(f)=R-\{0\}$

We have the following table of values:

| $x$ | -4 | -2 | -1 | -0.5 | -0.25 | 0.25 | 0.5 | 1 | 2 | 4 |
| :---: | :---: | :---: | :---: | :---: | :---: | :---: | :---: | :---: | :---: | :---: |
| $f(x)=\frac{1}{x}$ | -0.25 | -0.5 | -1 | -2 | -4 | 4 | 2 | 1 | 0.5 | 0.25 |

Now, we plot the points $A(-4,-0.25)$, $B(-2,-0.5)$, $C(-1,-1)$, $D(-0.5,-2)$, $E(-0.25,-4)$ and $F(0.25,4)$, $G(0.5,2)$, $H(1,1)$, $J(2,0.5)$ and $K(4,0.25)$. We join these points freehand successively to obtain the two curves of the hyperbola.
