### Answer to Question 04.

- [ ] InBoX/Ans/Ex/Class-09/Unit-6/Chapter-16/Section-Q/Q-04

- Answer: 

(b)

> [!INFO]
>
> Class size $=(20-15)=5$.
> 
> Class mark $=20$.
> 
> Lower limit $=\left(20-\frac{5}{2}\right)=\frac{35}{2}=17.5$
> 
> Upper limit $=\left(20+\frac{5}{2}\right)=\frac{45}{2}=22.5$
> 
> Required class is 17.5-22.5.
>

### Answer to Question 05.

- [ ] InBoX/Ans/Ex/Class-09/Unit-6/Chapter-16/Section-Q/Q-05

- Answer: 

(b)

> [!INFO]
>
> Let the lower limit be $a$. Then, upper limit $=a+6$.
> 
> Mid-value $=\frac{a+a+6}{2}=\frac{2 a+6}{2}=a+3$.
> 
> $\therefore \quad a+3=10 \Rightarrow a=7$.
> 
> $\therefore \quad$ lower limit of the class is 7.
>
