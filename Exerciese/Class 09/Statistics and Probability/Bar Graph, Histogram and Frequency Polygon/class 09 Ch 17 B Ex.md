### Answer to Question 06.

- [ ] InBoX/Ans/Ex/Class-09/Unit-6/Chapter-17/Section-B/Q-06

- Answer: 

> [!INFO]
>
> The class sizes are $4,6,12,20$ and 28 .
>
> Minimum class size $=4$.
> Adjusted frequency $=\frac{\text{ minimum class size }}{\text{ class size of this class }} \times$ its frequency.
> So, the adjusted frequencies of the given class intervals are
>
> $$
> \left(\frac{4}{4} \times 5\right)=5,\left(\frac{4}{6} \times 6\right)=4,\left(\frac{4}{12} \times 9\right)=3,\left(\frac{4}{20} \times 25\right)=5,\left(\frac{4}{28} \times 21\right)=3.
> $$
>
